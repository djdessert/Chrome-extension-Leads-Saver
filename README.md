# Chrome Extension - Leads Saver

==> This file set will create a Chrome extension to save web addresses that you type in, or can save the current address you're navigated to.

In order to install it: 
 - Head to chrome://extensions
 - Select Developer Mode in the upper right corner
 - Select 'Load Unpacked Extension' from the upper left button
 - Load this folder with all files (png, CSS, HTML, JS, JSONs)
 
 That's it! Once you navigate to your search bar, find the puzzle piece icon, select it and scroll to your new Leads Saver extension.
